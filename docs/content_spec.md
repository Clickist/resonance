## S6 — 综合画像

**综合所有信号给出性格推断**，包括：
- 执念式听众（30% 播了10次以上）
- 每年有新的深度挖掘触发点，通常来自番剧或偶然发现
- 不做显式评价（loved/rating全空），但播放行为本身极其主动
- 日系偏好明显但不固化，随番剧迁移
- 2020-2021是高密度发现期，之后节奏稳定，从未停止

## S7 — 推荐歌单

**三张可选歌单，用户三选一（或多选），确认后生成 AppleScript 创建播放列表**

1. **遗忘唤醒**：从122首从没播里，排除过时番剧热潮的，按当前偏好过滤推荐
2. **近期偏好推荐**：基于 Nujabes → 日系嘻哈 + SZA 主线，推同圈子艺人
3. **执念回顾**：播放次数最高的曲子重新整成一张单子

交互流程：展示曲目列表 → 用户确认 → 输出 AppleScript → 在 Claude Code 里执行
